# Scripts

This folder contains runnable scripts that demonstrate package usage outside Jupyter notebooks.

## `run_examples.py`

Runs a lightweight walkthrough that:

1. Compares classic and reduced Collatz steps for small integers.
2. Builds a `BaseCollatz` object for `17`.
3. Prints the decomposition expression and reconstructed value.

Run with:

```bash
python scripts/run_examples.py
```
